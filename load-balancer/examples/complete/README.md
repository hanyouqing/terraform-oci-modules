# Load Balancer Complete Example

This example demonstrates a production-ready load balancer setup.

## Features

- Flexible or fixed-shape load balancer
- Multiple backend sets with SSL/TLS
- Multiple listeners with SSL termination
- Network Security Group integration
- Health checks with custom configurations
- Comprehensive tagging

## Usage

```bash
terraform init
terraform plan
terraform apply
```

## Production Considerations

- SSL/TLS termination
- Health check optimization
- Network security with NSGs
- High availability configuration
- Performance tuning
